Client dan Server itu Apa ?

  • Server: komputer yg nyediain data buat komputer lain yg minta (Client), umumnya hidup 24 jam, terus-menerus terhubung ke jaringan kayak internet ato LAN. Biasa digunakan untuk penyimpanan data terpusat seluruh kegiatan dan pengguna aplikasi.
  • Client: komputer yg minta sesuatu ke Server. Kadang minta kirim data, kadang minta lihat data.

Tentang Server

  • Semua komputer bisa jadi server dengan menjalankan aplikasi Web Server
  • Aplikasi-aplikasi global seperti IG, Mobile Legends servernya biasanya ditaruh di Data Center yang jumlahnya bisa mencapai beribu-ribu
  • Di Server umumnya ada berbagai aplikasi Database, program terkait, dan aplikasi Web Server

Global State

Set data-state="something" on a slide and "something" will be added as a class to the document element when the slide is open. This lets you apply broader style changes, like switching the page background.

State Events

Additionally custom events can be triggered on a per slide basis by binding to the data-state name. 


Reveal.on( 'customevent', function() {
	console.log( '"customevent" has fired' );
} );
