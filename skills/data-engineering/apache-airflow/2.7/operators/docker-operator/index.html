How to Use DockerOperator in Apache Airflow 2.7 ?

## What is it ? DockerOperator is Apache Airflow operator for executing commands inside a docker container ## How to use ? ```py a = 4 my_task = DockerOperator( network="your_network_name", ) ``` ## Resources - [DockerOperator official documentation](https://airflow.apache.org/docs/apache-airflow-providers-docker/stable/_api/airflow/providers/docker/operators/docker/index.html)
